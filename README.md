# Lampolite
Lampolite is a lamp that runs when interacting with the touchpad for about 5.2 seconds 
# BOM ( Bill of material )
- 2 ICC timer NE555P
- 1 One expansion GIO pins 4017
- 20 LEDs
- 4 resistors
- 1 variable resistor
- 4 capacitors ( one is polarized )
- 6 pin headers : 3 for powering up the circuit and 3 for the TTP223 sensor
- The TTP223 Sensor
